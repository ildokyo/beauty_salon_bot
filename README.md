## О проекте

Beauty Salon Bot — это полнофункциональный Telegram-бот для автоматизации записи клиентов в салон красоты.  
Бот позволяет клиентам самостоятельно записываться на услуги, а администраторам — управлять расписанием, мастерами и услугами.

## ФУНКЦИИ БОТА: ВХОД И ВЫХОД

### КЛИЕНТСКИЕ КОМАНДЫ

| Команда | Вход (что вводит пользователь) | Выход (что выводит бот) |
|---------|-------------------------------|------------------------|
| `/start` | Ничего (отправка команды) | Приветственное сообщение и главное меню с кнопками |
| `/services` | Ничего | Список всех услуг: название, длительность, цена |
| `/masters` | Ничего | Список мастеров: имя, специализация, опыт работы |
| `/book` | Последовательный выбор: услуга → мастер → дата (ДД.ММ.ГГГГ) → время → имя → телефон | Подтверждение записи с указанием всех деталей |
| `/mybookings` | Ничего | Список активных записей с кнопкой "Отменить" |
| `/help` | Ничего | Справка по всем командам |

### УПРАВЛЕНИЕ МАСТЕРАМИ

| Действие | Вход | Выход |
|----------|------|-------|
| ➕ Добавить мастера | 1. Имя мастера<br>2. Специализация<br>3. Опыт работы (например: "5 лет")<br>4. Категория (`hair` или `nails`) | ✅ Мастер добавлен с указанными данными |
| 🗑 Удалить мастера | ID мастера (число из списка) | ✅ Мастер скрыт (мягкое удаление) |
| 🔄 Восстановить мастера | ID мастера (число из списка скрытых) | ✅ Мастер восстановлен |
| 📋 Список мастеров | Ничего | Список всех мастеров с ID, статусом и опытом |

---

### УПРАВЛЕНИЕ УСЛУГАМИ

| Действие | Вход | Выход |
|----------|------|-------|
| ➕ Добавить услугу | 1. Название (со смайликом)<br>2. Длительность в минутах<br>3. Цена в рублях<br>4. Категория (`hair` или `nails`) | ✅ Услуга добавлена |
| 🗑 Удалить услугу | ID услуги (число из списка) | ✅ Услуга скрыта |
| 🔄 Восстановить услугу | ID услуги (число из списка скрытых) | ✅ Услуга восстановлена |
| 📋 Список услуг | Ничего | Список всех услуг с ID, ценой, длительностью и статусом |

---

### ДОБАВЛЕНИЕ РАСПИСАНИЯ (автоматическая генерация)

| Шаг | Вход | Выход |
|-----|------|-------|
| 1 | ID мастера (число) | Запрос даты начала |
| 2 | Дата начала (ДД.ММ.ГГГГ), например `10.05.2026` | Запрос даты окончания |
| 3 | Дата окончания (ДД.ММ.ГГГГ), например `20.05.2026` | Запрос времени начала |
| 4 | Время начала (ЧЧ:ММ), например `10:00` | Запрос времени окончания |
| 5 | Время окончания (ЧЧ:ММ), например `18:00` | ✅ Отчёт о создании: количество дней, слотов, период |

**Результат:** Расписание создаётся на ВСЕ дни в указанном диапазоне (включая выходные) с интервалом 30 минут.

## БАЗА ДАННЫХ (SQLite)

### Таблицы и их поля

| Таблица | Поля | Описание |
|---------|------|----------|
| `clients` | client_id, telegram_id, name, phone, registered_at | Информация о клиентах |
| `masters` | master_id, name, specialization, experience, service_category, is_active | Мастера и их опыт |
| `services` | service_id, name, duration_min, price, category, is_active | Услуги с ценой и длительностью |
| `master_schedule` | schedule_id, master_id, work_date, time_start, time_end, is_booked, booking_id | Расписание мастеров |
| `bookings` | booking_id, client_id, master_id, service_id, schedule_id, booking_date, booking_time, status | Записи клиентов |
| `admins` | admin_id, telegram_id | Администраторы |

## УСТАНОВКА И ЗАПУСК

### 1. Клонирование репозитория

git clone https://github.com/ildokyo/beauty_salon_bot.git
cd beauty_salon_bot

### 2. Создание виртуального окружения

# Windows
python -m venv venv
venv\Scripts\activate

# Mac/Linux
python3 -m venv venv
source venv/bin/activate

### 3. Установка зависимостей

pip install -r requirements.txt

### 4. Настройка переменных окружения
Создай файл .env:

BOT_TOKEN=твой_токен_от_BotFather
ADMIN_SECRET_CODE=SALON2026

### 5. Запуск бота
bash
python bot.py
