# 💅 Телеграм-бот для салона красоты «Бабочка»

[![Python](https://img.shields.io/badge/Python-3.11-blue.svg)](https://www.python.org/)
[![Aiogram](https://img.shields.io/badge/aiogram-3.17-green.svg)](https://docs.aiogram.dev/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## 🤖 О проекте

Этот Telegram-бот предназначен для автоматизации записи клиентов в салон красоты. 
Бот позволяет клиентам просматривать услуги и мастеров, записываться на удобное время, 
а администраторам — управлять расписанием и персоналом.

### Основные возможности:

**Для клиентов:**
- 📋 Просмотр услуг и цен
- 👨‍🎨 Просмотр мастеров и их специализации
- 📅 Запись на услугу с выбором даты и времени
- 📋 Просмотр своих активных записей
- ❌ Отмена записи
- ℹ️ Информация о салоне и контакты

**Для администраторов:**
- 👑 Панель управления
- ➕ Добавление новых мастеров
- ➕ Добавление новых услуг
- 📅 Добавление рабочего расписания мастеров
- 📋 Просмотр всех записей клиентов
- 👥 Управление администраторами

## 🚀 Технологии

- **Python 3.11** — язык программирования
- **Aiogram 3.x** — фреймворк для Telegram Bot API
- **SQLite** — встроенная база данных
- **pytz** — работа с часовыми поясами
- **python-dotenv** — управление переменными окружения

## 📦 Установка и запуск (локально)

### 1. Клонировать репозиторий

git clone https://github.com/ildokyo/beauty_salon_bot.git
cd beauty_salon_bot

### 2. Создать виртуальное окружение
```
# Windows
python -m venv venv
venv\Scripts\activate
```
### 3. Установить зависимости
```
pip install -r requirements.txt
```
### 4. Настроить переменные окружения
Создай файл .env и добавь:
```
BOT_TOKEN=твой_токен_от_BotFather
ADMIN_SECRET_CODE=SALON2026
```
### 5. Запустить бота
```
python bot.py
```

### Команды бота
Для всех пользователей:
Команда	Описание
/start	Начать работу
/services	Показать услуги и цены
/masters	Показать мастеров
/book	Записаться на услугу
/mybookings	Мои записи
/help	Помощь

Для администраторов:
Команда	Описание
/becomeadmin КОД	Стать администратором (код: SALON2026)
/admin	Панель администратора
/add_master	Добавить мастера
/add_service	Добавить услугу
/add_schedule	Добавить расписание
/all_bookings	Все записи
/add_admin ID	Добавить администратора

### Структура проекта

beauty_salon_bot/
├── bot.py              # Главный файл запуска
├── config.py           # Настройки и переменные окружения
├── database.py         # Работа с базой данных
├── handlers.py         # Обработчики клиентских команд
├── admin_handlers.py   # Обработчики административных команд
├── keyboards.py        # Клавиатуры и кнопки
├── utils.py            # Вспомогательные функции
├── requirements.txt    # Зависимости проекта
├── .env               # Переменные окружения (не в Git)
└── README.md          # Документация

### База данных
Бот использует SQLite с следующими таблицами:

clients — информация о клиентах
masters — список мастеров
services — услуги и цены
master_schedule — рабочее расписание мастеров
bookings — записи клиентов
admins — администраторы
